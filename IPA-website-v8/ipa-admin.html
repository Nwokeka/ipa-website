Update Admin Password

⚠️ After changing the password, immediately share the new one with all current executives. The old team will no longer be able to log in.

💡 Best Practice for Election Changeover

1. Incoming President or Secretary logs in with old password
2. Goes to this page and sets a new strong password
3. Shares new password securely with new executive team only
4. Old executives lose access immediately

Recommended: change password within 48 hours of election results.
